Тестовое задание на позицию UI Developer/Integrator

Требуется:
 - Создать чистый проект. Версия Unity 2018 LTS
 - Установить TextMeshPro если он не установлен (далее для текста нужно использовать его)
   Шрифт можно использовать стандартный из TextMeshPro
 - Сверстать окно используя реф материалы и исходную графику
 - необходимо использовать UI систему Unity (RectTransform, Canvas etc.)
   значения компонента CanvasScaler для должны быть следующие:
   UI Scale Mode: Scale With Screen Size
   Reference Resolution: X=2436 Y=1125
   Screen Match Mode: Match Width Or Height
   Match: 0.5
   Reference Pixels Per Unit: 100
 - разработать тестовые скрипты для него (описания ниже)
 - сформировать список рекомендаций по улучшению/оптимизации графики

Необходимые скрипты:
1. Кнопка крестик останавливает PlayMode в Unity
2. При каждом появлении окна должна происходить какая-нибудь анимация, например скейлить иконку пушки. Или любая другая по вкусу.
3. Желтая кнопка должна начислять пончики (с анимацией)
4. Значение прогресс бара и кол-во звездочек должно быть рандомно при каждом появлении окна

Ссылка на рефенс:
 https://www.figma.com/file/vMIM3QucDujcF6aTsfXaXD/Zombie-Test?node-id=0%3A1

При выполнении задания стоит обращать внимание на следующее:
- именование
  Названия файлов, ассетов, GameObject'ов и прочего должны быть в одном стиле и иметь логический смысл.
- структура проекта
  Расположение всех элементов проекта должно быть интуитивно понятным
- внимание к деталям исходных графических материалов
  При интеграции графики, важно учитывать все детали референсов
- Оптимизация
  Следует учитывать все особенности мобильной разработки и оптимизации графики под них
